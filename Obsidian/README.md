# BQB 학습 자료 - 옵시디언 사용 가이드

## 📚 포함된 파일

이 저장소에는 4개의 HTML 학습 자료가 포함되어 있습니다:

1. **bqb-beginner.html** - BQB 초보자 가이드
2. **bqb-intermediate.html** - BQB 중급 가이드
3. **quantum-world.html** - Quantum World Explorer
4. **bqb-advanced.html** - BQB 고급 가이드

---

## 🚀 옵시디언에서 보는 방법

### 1단계: HTML Reader 플러그인 설치

1. 옵시디언을 실행합니다
2. **설정(⚙️)** → **커뮤니티 플러그인** 으로 이동
3. **제한 모드 끄기**를 클릭 (보안 경고 확인)
4. **찾아보기** 버튼 클릭
5. 검색창에 **"HTML Reader"** 입력
6. **설치(Install)** 클릭 후 **활성화(Enable)** 클릭

---

### 2단계: HTML 파일을 옵시디언 Vault에 복사

#### 방법 A: 직접 복사 (추천)

1. 이 저장소에서 4개 HTML 파일을 다운로드
2. 옵시디언 Vault 폴더를 엽니다
   - Windows: `C:\Users\[사용자명]\Documents\[Vault명]\`
   - Mac: `~/Documents/[Vault명]/`
3. 4개 파일을 Vault 폴더에 복사합니다

#### 방법 B: Git Clone

```bash
# Vault 폴더로 이동
cd ~/Documents/MyVault

# 저장소 클론
git clone [이 저장소 URL]

# 또는 파일만 복사
cp [저장소경로]/*.html .
```

---

### 3단계: 파일 열기

1. 옵시디언 왼쪽 사이드바에서 HTML 파일 찾기
2. 파일을 **클릭**하면 자동으로 렌더링됩니다
3. 또는 **우클릭** → **Open with HTML Reader** 선택

---

## 💡 사용 팁

### 추천 학습 순서

```
1. bqb-beginner.html    ← 여기서 시작!
   ↓
2. quantum-world.html   ← 양자 개념 이해
   ↓
3. bqb-intermediate.html ← 기술 심화
   ↓
4. bqb-advanced.html    ← 프로젝트 전체상
```

### 키보드 단축키

- **Ctrl/Cmd + Click**: 새 탭으로 열기
- **Ctrl/Cmd + W**: 탭 닫기
- **Ctrl/Cmd + Tab**: 탭 전환

### 검색 활용

옵시디언 검색(Ctrl/Cmd + Shift + F)으로 HTML 파일 내용도 검색 가능합니다:

- `σ_z` - 종방향 산란 관련 내용 찾기
- `Triple-Wall` - 3중 보호벽 내용 찾기
- `큐비트` - 큐비트 관련 모든 내용

## ⚙️ 문제 해결

### HTML 파일이 보이지 않아요

1. HTML Reader 플러그인이 **활성화**되어 있는지 확인
2. 파일 확장자가 `.html`인지 확인
3. 옵시디언을 재시작해보세요

### 애니메이션이 작동하지 않아요

- 정상입니다! 모든 애니메이션은 CSS 기반이며 자동 재생됩니다
- 브라우저가 아닌 옵시디언에서는 일부 애니메이션이 제한될 수 있습니다

### 한글이 깨져요

- 파일 인코딩이 UTF-8인지 확인하세요
- 이 파일들은 모두 UTF-8로 인코딩되어 있습니다

### 모바일에서 보고 싶어요

1. 옵시디언 모바일 앱 설치
2. Vault 동기화 (iCloud, Dropbox 등)
3. HTML Reader 플러그인 설치 (모바일에서도 가능)
